# Food Delivery Portfolio App

Expo + TypeScript portfolio app scaffold for a mock food delivery experience.

## Stack

- Expo managed workflow
- TypeScript
- React Navigation
- Zustand
- react-native-reanimated
- expo-google-fonts

## Scripts

- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`
- `npm run typecheck`
